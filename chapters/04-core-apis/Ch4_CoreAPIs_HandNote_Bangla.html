SECTION 01

String তৈরি ও পরিবর্তন

একটা String হলো character-এর sequence এবং একটা reference type। এটা বিশেষ — এর জন্য new লাগে না। নিচের তিনটাই "Fluffy" নামের একটা String তৈরি করে:

String name = "Fluffy";
String name = new String("Fluffy");
String name = """
Fluffy""";  // text block

String হলো CharSequence-এর implementation (StringBuilder-ও তাই)। String immutable — তাই প্রতিটা "modifying" method আসলে একটা নতুন String return করে।

Concatenation

Concatenation-এর তিনটা নিয়ম — মুখস্থ রাখুন

  1. দুটো operand-ই numeric হলে + মানে সংখ্যার যোগ।
  2. যেকোনো একটা operand String হলে + মানে concatenation।
  3. Expression-টা বাঁ থেকে ডানে মূল্যায়ন হয়।
System.out.println(1 + 2);            // 3
System.out.println("a" + "b");        // ab
System.out.println("a" + "b" + 3);    // ab3
System.out.println(1 + 2 + "c");      // 3c
System.out.println("c" + 1 + 2);      // c12
System.out.println("c" + null);        // cnull
⚠ EXAM TRAP

mixed expression-এ variable-এর type খেয়াল করুন। যদি int three = 3; আর String four = "4"; হয়:

  • 1 + 2 + three + four
  • 1 + 2 → দুটোই int → 3
  • 3 + three → দুটোই int → 6
  • 6 + four → four একটা String → concat-এ চলে যায় → "64"

+= একই নিয়ম মানে: s += "2" মানে s = s + "2"। var s = "1" থেকে শুরু করে s += "2" → "12", তারপর s += 3 → "123" (String-এর সাথে যা-ই যোগ হোক → concatenation)।

গুরুত্বপূর্ণ String method

Index zero-based, কিন্তু length() সাধারণ গোনার নিয়মে চলে। "animals"-এর জন্য: length 7, valid index 0–6।

length() ও charAt(int)

var name = "animals";
name.length();    // 7
name.charAt(0);   // a
name.charAt(6);   // s
name.charAt(7);   // StringIndexOutOfBoundsException

indexOf(...) — না পেলে -1 ফেরায় (exception দেয় না)

start index inclusive, end index exclusive। একটা char (int ch হিসেবে) বা একটা String নিতে পারে।

var name = "animals";
name.indexOf('a');       // 0
name.indexOf("al");      // 4
name.indexOf('a', 4);    // 4
name.indexOf("al", 5);   // -1 (search starts after the match)
name.indexOf('a', 2, 4); // -1 (only looks at indexes 2,3)
name.indexOf("al", 2, 6);// 4

substring(begin[, end]) — পরীক্ষার সবচেয়ে ট্রিকি method

✓ KEY RULE

begin (inclusive) থেকে end (exclusive) পর্যন্ত character ফেরায়। end index length-এর সমান (অর্থাৎ এক ঘর বেশি) হতে পারে, কিন্তু উল্টানো (reversed) যাবে না, আর length-এর থেকে একের বেশি বেশি হওয়া যাবে না।

var name = "animals";  // length 7
name.substring(3);     // mals
name.substring(3, 4);  // m
name.substring(3, 7);  // mals  (7 == end, allowed)
name.substring(3, 3);  // "" empty string
name.substring(3, 2);  // exception (reversed)
name.substring(3, 8);  // exception (8 > length)

Case, search, replace, whitespace

Methodআচরণ (উল্লেখ না থাকলে case-sensitive)
toUpperCase() / toLowerCase()অক্ষর convert করে; বাকি character অপরিবর্তিত রাখে।
equals(Object) / equalsIgnoreCase(String)equals একটা Object নেয়; non-String দিলে → false। "ABC".equals(6) → false।
startsWith(p[,from]) / endsWith(s) / contains(CharSequence)case-sensitive prefix / suffix / যেকোনো জায়গায় check।
replace(char,char) / replace(CharSequence,CharSequence)সহজ search ও replace, নতুন String ফেরায়।
strip() / trim()সামনে-পেছনের whitespace বাদ দেয়। strip() Unicode-aware; trim() নয়।
stripLeading() / stripTrailing()শুধু এক পাশের whitespace বাদ দেয়।

String text = " abc\t "-এর জন্য (সামনে space + "abc" + tab + space):

  • text.trim().length() → 3
  • text.strip().length() → 3
  • text.stripLeading().length() → 5 (পেছনের tab + space থাকে)
  • text.stripTrailing().length() → 4 (সামনের space থাকে)
★ REMEMBER

পরীক্ষায় whitespace = space, \t (tab), \n (newline) এবং \r (carriage return)। \t হলো একটাই character।

Indentation: indent(n) ও stripIndent()

MethodIndent পরিবর্তনআগের line break normalize করে?না থাকলে শেষে line break যোগ করে?
indent(n), n > 0প্রতি line-এ n টা space যোগহ্যাঁহ্যাঁ
indent(n), n == 0পরিবর্তন নেইহ্যাঁহ্যাঁ
indent(n), n < 0প্রতি line থেকে n পর্যন্ত space বাদহ্যাঁহ্যাঁ
stripIndent()সব leading incidental whitespace বাদহ্যাঁনা
⚠ EXAM TRAP — সেই "extra" character

যেহেতু indent() line termination normalize করে, trailing newline না থাকা একটা text block একটা newline পেয়ে যায়। একটা 6-character block indent(1)-এর পর 10 হয় (3 টা space যোগ + 1 normalized newline), 9 নয়।

Empty বনাম blank

" ".isEmpty();  // false (has a space)
"".isEmpty();   // true
" ".isBlank();  // true (only whitespace)
"".isBlank();   // true

Formatting

String.format(...) static; formatted(...) instance method। যে symbol-গুলো জানতেই হবে:

Symbolকীসের জন্য
%sযেকোনো type (সাধারণত String)
%dInteger মান (int, long)
%fFloating-point (float, double) — default-এ 6 দশমিক, round করে
%nsystem-নির্ভর line break
⚠ EXAM TRAP

type না মিললে runtime-এ exception: "Food: %d tons".formatted(2.0) → IllegalFormatConversionException (একটা double দেওয়া হলো %d-তে)। %.1f দিয়ে 90.250000 প্রিন্ট হয় 90.3 (round, truncate নয়)।

Method chaining

বাঁ থেকে ডানে পড়ুন; আগের call-এর return value-এর উপর পরের method call হয়। immutable String-এ প্রতিটা call একটা নতুন String বানায়।

  • "AniMaL ".trim().toLowerCase().replace('a','A')
  • trim() → "AniMaL"
  • toLowerCase() → "animal"
  • replace('a','A') → "AnimAl"

যেহেতু String immutable, মূল variable কখনো বদলায় না:

String a = "abc";
String b = a.toUpperCase();          // b = "ABC", a still "abc"
b = b.replace("B", "2").replace('C', '3');
// "ABC" -> "A2C" -> "A23"; a="abc", b="A23"
SECTION 02

StringBuilder Class

একটা loop-এ String-এ append করলে অনেক throwaway object তৈরি হয় ("" + 26 অক্ষর → 27টা String object)। StringBuilder mutable এবং একটাই object বারবার ব্যবহার করে।

String

  • Immutable
  • Method নতুন String ফেরায়
  • Chained call নতুন object বানায়
  • Pooled (literal/constant)

StringBuilder

  • Mutable
  • বেশিরভাগ method mutate করে ও this ফেরায়
  • Chained call একই object-এ কাজ করে
  • Pooled নয়; StringBuffer হলো thread-safe (ধীর) cousin — পরীক্ষায় নেই

Mutability ও chaining

⚠ EXAM TRAP

StringBuilder chain করলে নতুন object তৈরি হয় না — একই object-কে mutate করে এবং তার reference ফেরায়। তাই নিজের উপর assignment করলে "কিছুই হয় না"।

StringBuilder a = new StringBuilder("abc");
StringBuilder b = a.append("de");       // a and b -> "abcde"
b = b.append("f").append("g");        // same object -> "abcdefg"
System.out.println("a=" + a);          // a=abcdefg
System.out.println("b=" + b);          // b=abcdefg

StringBuilder তৈরি করা

new StringBuilder();        // empty
new StringBuilder("animal"); // initial value
new StringBuilder(10);      // reserve capacity 10 (not contents!)

গুরুত্বপূর্ণ method

String-এর মতোই common method: charAt, indexOf, length, substring

✓ KEY RULE

substring() একটা String ফেরায় এবং StringBuilder-কে mutate করে না — শুধু state সম্পর্কে জানতে চায়। তুলনায় append(), insert(), delete() mutate করে।

var sb = new StringBuilder("animals");
String sub = sb.substring(sb.indexOf("a"), sb.indexOf("al")); // substring(0,4)="anim"
int len = sb.length();   // 7
char ch = sb.charAt(6);  // s
System.out.println(sub + " " + len + " " + ch); // anim 7 s

append — 10+ overload, this ফেরায়

var sb = new StringBuilder().append(1).append('c');
sb.append("-").append(true);
System.out.println(sb); // 1c-true

insert(int offset, ...) — যত এগোয় index তত shift হয়

var sb = new StringBuilder("animals");
sb.insert(7, "-");  // animals-
sb.insert(0, "-");  // -animals-
sb.insert(4, "-");  // -ani-mals-

delete(start,end) ও deleteCharAt(index)

delete() উদার: length-এর বেশি end দিলে সেটাকে length ধরে নেয়। deleteCharAt() কড়া।

var sb = new StringBuilder("abcdef");
sb.delete(1, 3);       // adef
sb.deleteCharAt(5);   // exception (only 4 chars left)

var sb2 = new StringBuilder("abcdef");
sb2.delete(1, 100);    // a  (end past length is OK)

replace(start,end,str) — String-এর replace থেকে আলাদা!

প্রথমে যুক্তিগতভাবে [start,end) delete করে, তারপর নতুন text insert করে (length মেলার দরকার নেই; length-এর বেশি end = end)।

var b = new StringBuilder("pigeon dirty");
b.replace(3, 6, "sty"); // pigsty dirty

var c = new StringBuilder("pigeon dirty");
c.replace(3, 100, ""); // pig

reverse() ও toString()

var sb = new StringBuilder("ABC");
sb.reverse();           // CBA
String s = sb.toString(); // convert back to String
⚠ EXAM TRAP — chained inserts
  • new StringBuilder().append("aaa").insert(1,"bb").insert(4,"ccc")
  • append → aaa
  • insert(1,"bb") → abbaa
  • insert(4,"ccc") → abbaccca
SECTION 03

Equality বোঝা

== (reference)

  • Primitive: মান তুলনা করে
  • Object: মেমরিতে একই object কি না?
  • StringBuilder-এ: reference check

equals() (logical)

  • class-এর implementation-এর উপর নির্ভর করে
  • String.equals: একই character?
  • StringBuilder equals override করে না → reference check
var one = new StringBuilder();
var two = new StringBuilder();
var three = one.append("a");   // append returns 'one'
System.out.println(one == two);   // false (different objects)
System.out.println(one == three); // true  (same object)
⚠ EXAM TRAP — type mismatch won't compile 
var name = "a";
var builder = new StringBuilder("a");
System.out.println(name == builder); // DOES NOT COMPILE

Compiler জানে যে সম্পূর্ণ আলাদা দুটো type কখনো একই object reference করতে পারে না।

String pool

✓ KEY RULE

String pool-এ থাকে literal এবং compile-time constant। runtime-এ হিসাব করা যেকোনো কিছু (trim()-এর মতো method call, +=, বা new String(...)) pool-এর বাইরে একটা নতুন object বানায়।

var x = "Hello World";
var y = "Hello World";
System.out.println(x == y);             // true  (same pooled literal)

var z = " Hello World".trim();
System.out.println(x == z);             // false (computed at runtime)

var concat = "hello "; concat += "world";
System.out.println("hello world" == concat); // false (+= -> new String)

var n  = "Hello World";
var n2 = new String("Hello World").intern();
System.out.println(n == n2);            // true  (intern() pulls from pool)

Compile-time constant বনাম runtime

  • first = "rat" + 1 → compile-time constant "rat1" (pooled)
  • second = "r"+"a"+"t"+"1" → এটাও compile-time constant → একই pool ref
  • third = "r"+"a"+"t"+new String("1") → constant নয় → নতুন object
  • first == second → true
  • first == second.intern() → true
  • first == third → false
  • first == third.intern() → true (intern pool-এ "rat1" খুঁজে পায়)
★ REMEMBER

বাস্তব কোডে String তুলনা করতে সবসময় equals() ব্যবহার করুন — কখনো == বা intern() নয়। ওগুলো শুধু পরীক্ষাতেই দরকারি।

SECTION 04

Array বোঝা

Array হলো heap-এ থাকা একটা fixed-size, ordered list; এতে duplicate ও যেকোনো Java type রাখা যায়। char[] একটা reference variable; কিন্তু এর element-গুলো char primitive।

Primitive-এর array তৈরি

int[] numbers = new int[3];          // all default 0
int[] more    = new int[]{42, 55, 99};
int[] shorter = {42, 55, 99};         // anonymous array

নিচের পাঁচটাই একই জিনিস declare করে — bracket আগে বা পরে, space থাকুক বা না থাকুক:

int[] a;  int [] b;  int []c;  int d[];  int e [];
⚠ EXAM TRAP — এক line-এ একাধিক variable হলে bracket-এর অবস্থান 
int[] ids, types;  // BOTH are int[]
int ids[], types;  // ids is int[]; types is a plain int!

Reference type-এর array

Array হলো object; equals() override করা নেই, তাই এটা reference equality ব্যবহার করে (কখনো element তুলনা করে না)।

String[] bugs = {"cricket", "beetle", "ladybug"};
String[] alias = bugs;
String[] other = {"cricket", "beetle", "ladybug"};
bugs.equals(alias);  // true  (same reference)
bugs.equals(other);  // false (different object, elements ignored)
bugs.toString();     // [Ljava.lang.String;@... (use Arrays.toString instead)

Arrays.toString(bugs) → [cricket, beetle, ladybug]। initialize না করা array field হয় null; new String[2] দিলে দুটো null slot পাওয়া যায়।

Array casting ও ArrayStoreException

⚠ EXAM TRAP — compile হয় কিন্তু runtime-এ throw করে 
String[] strings = {"stringValue"};
Object[] objects = strings;           // widening, no cast needed
String[] again = (String[]) objects;   // narrowing, cast needed
again[0] = new StringBuilder();        // DOES NOT COMPILE (String[] needs String)
objects[0] = new StringBuilder();      // compiles, but throws ArrayStoreException at runtime

Array ব্যবহার — length একটা field

String[] mammals = {"monkey", "chimp", "donkey"};
mammals.length;     // 3 (no parentheses!)
mammals.length();   // DOES NOT COMPILE
new String[6].length; // 6 (slots, even if all null)
⚠ EXAM TRAP — ArrayIndexOutOfBoundsException

size 10-এর একটা array-তে valid index 0–9। নিচের তিনটাই runtime-এ throw করে:

numbers[10] = 3;               // index 10 invalid
numbers[numbers.length] = 5;   // length is one past max index
for (int i = 0; i <= numbers.length; i++) ... // <= reaches index 10

Sorting

Arrays.sort(array) ascending sort করে। String alphabetically sort হয়, তাই digit-string চমকে দিতে পারে।

Arrays.sort(new int[]{6,9,1});         // 1 6 9
Arrays.sort(new String[]{"10","9","100"}); // 10 100 9 (alphabetic!)
★ REMEMBER — "7Up" sort order

সংখ্যা (7) আগে sort হয়, তারপর Uppercase (U), তারপর lowercase (p)।

Arrays.binarySearch দিয়ে search

Binary search-এর নিয়ম (array অবশ্যই sorted হতে হবে)

  • পেলে: match-এর index ফেরায়।
  • না পেলে: -(insertion point) - 1 ফেরায় (একটা negative সংখ্যা)।
  • Unsorted array: ফলাফল undefined — উত্তর হিসেবে "unpredictable" বেছে নিন।
int[] n = {2,4,6,8};
Arrays.binarySearch(n, 2);  // 0
Arrays.binarySearch(n, 4);  // 1
Arrays.binarySearch(n, 1);  // -1  (insert at 0 -> -(0)-1)
Arrays.binarySearch(n, 3);  // -2  (insert at 1 -> -(1)-1)
Arrays.binarySearch(n, 9);  // -5  (insert at 4 -> -(4)-1)

Array তুলনা: equals / compare / mismatch

MethodArray-তে একই ডেটাArray আলাদা
Arrays.equals()truefalse
Arrays.compare()0positive বা negative সংখ্যা
Arrays.mismatch()-1প্রথম পার্থক্যের index (≥0)

compare()-এর ordering নিয়ম

  • Negative = প্রথম array "ছোট"; 0 = সমান; positive = প্রথমটা "বড়"।
  • সব leading element মিললে ছোট array-টাই ছোট।
  • null < যেকোনো মান; সংখ্যা < অক্ষর; uppercase < lowercase; prefix ছোট।
  • আলাদা type-এর array তুলনা → DOES NOT COMPILE।
Arrays.mismatch(new int[]{1}, new int[]{1});      // -1 (equal)
Arrays.mismatch(new String[]{"a"}, new String[]{"A"}); // 0  (differ at index 0)
Arrays.mismatch(new int[]{1,2}, new int[]{1});  // 1  (extra element at index 1)

Varargs ও multidimensional array

একটা varargs parameter (String... args) method-এর ভিতরে সাধারণ array হিসেবেই ব্যবহার হয় (args.length, args[0])। তিনটা main signature-ই valid: String[] args, String args[], String... args।

int[][] vars1;          // 2D
int vars2[][];          // 2D
int[] vars3[];          // 2D
int[] vars4[], space[][];// vars4 is 2D, space is 3D

String[][] rect = new String[3][2];        // rectangular
int[][] jagged = {{1,4}, {3}, {9,8,7}};   // asymmetric
int[][] partial = new int[2][];          // first dim only; fill later
✓ KEY RULE

array-of-array-এ পরের dimension-গুলোর size বাদ দেওয়া যায়, কিন্তু new ব্যবহার করলে প্রথম dimension-এর size দিতেই হবে (যেমন new int[2][] বৈধ, new int[][] নয়)।

BigInteger ও BigDecimal

মান primitive-এ না আঁটলে বা precision জরুরি হলে (যেমন টাকা) এগুলো ব্যবহার করুন। দুটোই immutable — তাই method chain করুন। valueOf() preferable; long দুটোতেই দেওয়া যায়, কিন্তু double শুধু BigDecimal-এ। Constant আছে যেমন BigInteger.ZERO, BigDecimal.ONE।

Date ও time তৈরি

✓ KEY RULE — মাস 1 থেকে গোনা হয়

index-এর মতো নয়, Date-Time API মাস গোনে 1 থেকে (মানুষের মতোই)। Month.JANUARY == 1।

LocalDate.of(2025, Month.JANUARY, 20);  // same as below
LocalDate.of(2025, 1, 20);
LocalTime.of(6, 15);            // hour, minute
LocalTime.of(6, 15, 30);        // + seconds
LocalTime.of(6, 15, 30, 200);   // + nanoseconds
LocalDateTime.of(date, time);     // combine
ZonedDateTime.of(dateTime, ZoneId.of("US/Eastern"));
⚠ EXAM TRAP — public constructor নেই

এই class-গুলো factory pattern ব্যবহার করে (private constructor + static method):

var d = new LocalDate();                       // DOES NOT COMPILE
var d = LocalDate.of(2025, Month.JANUARY, 32); // DateTimeException (no Jan 32)

আরও খেয়াল: ZonedDateTime.of-এ Month enum নেওয়া কোনো overload নেই।

পরিবর্তন — immutable, তাই return value ব্যবহার করুন

⚠ EXAM TRAP — উপেক্ষিত return value 
var date = LocalDate.of(2025, Month.JANUARY, 20);
date.plusDays(10);            // result thrown away!
System.out.println(date);    // still 2025-01-20
  • শুরু 2025-01-20
  • plusDays(2) → 2025-01-22
  • plusWeeks(1) → 2025-01-29
  • plusMonths(1) → 2025-02-28 (2025-এ Feb 29 নেই)
  • plusYears(5) → 2030-02-28
★ REMEMBER — leap year

Leap year হলো 4 বা 400-এর গুণিতক, কিন্তু 100-এর অন্য গুণিতক নয়। তাই 2000 ও 2028 leap year; 2100 নয়।

⚠ EXAM TRAP — type-এর জন্য ভুল unit 
var date = LocalDate.of(2025, Month.JANUARY, 20);
date = date.plusMinutes(1); // DOES NOT COMPILE (LocalDate has no time)

একটা field বদলে copy করতে with...() method (যেমন withYear, withDayOfMonth), আর type পরিবর্তন করতে at...() method (atTime, atStartOfDay, atDate, atZone) ব্যবহার করুন।

Period (date-based)

  • দিন, মাস, বছর

  • ofYears/ofMonths/ofWeeks/ofDays/of(y,m,d)

  • LocalTime-এ নয়

Duration (time-based)

⚠ EXAM TRAP — Period factory chain করা যায় না 
var wrong = Period.ofYears(1).ofWeeks(1); // only ofWeeks(1) is used!

এগুলো static method, তাই chain করলে শুধু overwrite হয়। of() শুধু year/month/day নেয়। Period.ofMonths(3) প্রিন্ট করে P3M; Period.between(xmas, newYears) → P7D, উল্টো দিকে → P-7D।

Instant ও daylight saving time

Instant হলো GMT-তে একটা মুহূর্ত। toInstant() দিয়ে একটা ZonedDateTime-কে Instant-এ রূপান্তর করা যায়, কিন্তু একটা LocalDateTime-কে যায় না (time zone নেই → কোনো নির্দিষ্ট মুহূর্ত নেই)।

U.S. daylight saving time (পরীক্ষায় শুধু এটাই আসে)

  • Spring forward (মার্চ, ~2 a.m.): 1:59 a.m. সরাসরি 3:00 a.m.-এ লাফ দেয় — 2:30 a.m. বলে কিছু নেই। Offset বদলায় -05:00 → -04:00।
  • Fall back (নভেম্বর, ~2 a.m.): 1:00–1:59 a.m. ঘণ্টাটা দুবার ঘটে। Offset বদলায় -04:00 → -05:00।
  • যে time-টা থাকে না সেটা বানালে (যেমন spring-forward দিনে 2:30) সেটা সোজা সামনে রোল করে 3:30-এ যায়।
  • March 9, 2025, 1:30 EST → offset -05:00, hour 1
  • plusHours(1) → 3:30, offset -04:00, hour 3
  • GMT check: 1:30 −(−5:00)=6:30 GMT → 3:30 −(−4:00)=7:30 GMT (ঠিক এক ঘণ্টা)
★ SHORT RECAP

সংক্ষিপ্ত রিভিউ

Strings

  • String immutable; দুটোই numeric হলে + = যোগ, নয়তো concatenation, বাঁ→ডান মূল্যায়ন।
  • substring(b,e): b inclusive, e exclusive; e length-এর সমান হতে পারে কিন্তু বেশি বা reversed নয়; না পেলে indexOf ফেরায় -1।

StringBuilder

  • Mutable; append/insert/delete mutate করে ও this ফেরায়; substring mutate না করে একটা String ফেরায়।
  • delete/replace length-এর বেশি end index মেনে নেয়; deleteCharAt নেয় না।

Equality

  • == reference equality; String.equals character তুলনা করে; StringBuilder equals override করে না।
  • Literal ও compile-time constant pooled; runtime-এ বানানো string ও new String() নয়; intern() pool থেকে টেনে আনে।

Arrays

  • length একটা field (parenthesis নেই); index 0..length-1; array-তে equals reference-ভিত্তিক।
  • sorted array-তে binarySearch index বা -(insertion point)-1 ফেরায়; unsorted → undefined।

Math

  • return type খেয়াল করুন: round(double)→long, round(float)→int, pow→double, random→double, ceil/floor→double।
  • long-এ না আঁটা literal compile হয় না; BigInteger/BigDecimal ব্যবহার করুন (immutable, valueOf)।

Dates & Times

  • মাস 1 থেকে গোনা; class immutable ও private constructor; উপেক্ষিত return value একটা ক্লাসিক trap।
  • Period = date unit (LocalTime নয়); Duration = time unit (LocalDate নয়); ভুল type → UnsupportedTemporalTypeException।
