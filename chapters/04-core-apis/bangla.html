কোর API নিয়ে কাজ

জাভাতে String হলো immutable — একবার তৈরি হলে এর ভেতরের ক্যারেক্টার ডেটা আর বদলায় না। যেসব মেথড দেখে মনে হয় স্ট্রিং বদলে দিচ্ছে, সেগুলো আসলে একটি নতুন অবজেক্ট রিটার্ন করে — এই কারণেই কম্পাইলার নিরাপদে একই লিটারেল পুল করতে পারে। 

var name = "OCP";
String greeting = "Hello, " + name;

System.out.println(greeting.length());      // 12
