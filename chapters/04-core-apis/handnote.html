Working with Core APIs

Strings in Java are immutable — once a String is created, the character data behind it never changes. Every method that looks like it edits a string actually returns a brand-new object, which is exactly why the compiler can safely pool identical literals. 

var name = "OCP";
String greeting = "Hello, " + name;

System.out.println(greeting.length());      // 12
System.out.println(greeting.indexOf("O"));  // 7
NOTE · মনে রাখুন

Prefer StringBuilder when concatenating inside a loop — it mutates one buffer instead of allocating a fresh String each pass.
